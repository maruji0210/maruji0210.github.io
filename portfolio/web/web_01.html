注文の多い料理店

Planning, Logo Design, Character Design, Web Design, Coding

誰もが知っている普遍的なストーリーである「注文の多い料理店」を効果的なパララックスという Webならではの表現で展開。「ユーザー自身が読み進めるという点で普及されているWeb動画との差別化を図る。ストーリーを8分割し、場面ごとにイラストのテイストを変化させることで、作品が持つ異世界感を演出し、且つコミカルなWeb表現を狙う。

物語に8つの部屋に入るシーンがあること、制作メンバーが9人いたことため、1部屋のシーンを各々1人〜2人に分かれ担当し、8つの部屋ごとに異なるテイストで制作し、物語の世界観に合う作品に仕上げた。

ただし、キャラクターの特徴に共通点がないとユーザーが読み進めるたびに別のキャラクターと困惑させぬために、キャラクター設定を共有した。
