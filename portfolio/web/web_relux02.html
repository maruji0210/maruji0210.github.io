08

Relux Campaign

Web Design, Coding

一流ホテル・旅館の宿泊予約サイトReluxのキャンペーンのデザイン・コーディングとSNSで拡散するOGP画像、メルマガ、LINEリッチコンテンツ等のシェア画像を作成。
上記はReluxサービス開始4周年を記念し、総額4億円相当の旅行クーポンが当たるキャンペーン「Relux4周年祭」の日本語版・繁体字版のPC・SPページをデザイン、コーディングを担当。 キャンペーンチームと相談しながら、デザイン35案の中からキャンペーン時期が春さきだったということもあり、桜の舞いで祭り感を出したデザイン案となった。
従来と異なり、エントリーボタンを固定fotterにするなど試みたことで従来のキャンペーンより会員数、予約数が大幅に増えた。

https://rlx.jp/cp/4thyear

*現在キャンペーン期間終えているため、閉鎖中。

春のさきどりキャンペーンの日本語版・繁体字版のPC・SPページをデザイン、コーディングを担当。
キャンペーン時期がまだ春と呼ぶには早い時期のため、春が芽吹く季節写真を選び、キャッチコピーにあるクーポン訴求を抑えたメインビジュアルに仕上げた。 また、今回のみキャンペーン時期が旧正月と重なるため、繁体字版のメインビジュアルデザインを旧正月のイメージに合う天燈の写真を選び仕上げた。

Data - 2016.09~2017.04 計7回のCPページを作成
